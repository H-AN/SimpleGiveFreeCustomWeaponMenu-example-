---
此插件创建一个菜单,并将武器密码指令填入配置文件,设置菜单显示名称

当玩家输入!weapon后打开菜单,选择项目,之后服务器将自动为玩家发送隐藏指令,并让玩家获取自定义武器.

这只是一个简单的使用示例,你可以根据例子配置 shop或者vip 权限之类的用途

---
This plugin creates a menu, fills in weapon password commands in a configuration file

sets the menu display name, and when a player types "!weapon", the menu opens, they select an item

and the server automatically sends a hidden command to the player, granting them a custom weapon. 

This is just a simple usage example; you can configure it for purposes such as shop or VIP access based on this example.
